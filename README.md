# Digital Clock using Python

## Description

This is a simple digital clock application built using Python and Tkinter. It shows the current time and date in real time and updates every second. 
I created this project to understand how GUI applications work in Python and how to handle time-based updates.
 

## Features

* Displays current time (HH:MM:SS)
* Shows AM/PM format
* Displays current date
* Updates automatically every second
* Simple and clean GUI



## Tech Used

* Python
* Tkinter (for GUI)
* time module



## How to Run

1. Make sure Python is installed on your system

2. Clone the repository
   git clone https://github.com/your-username/digital-clock.git

3. Go to the project folder
   cd digital-clock

4. Run the file
   python clock.py
